Custom Clocks

Custom Clocks gives you the ability to create multiple clocks customizable. Each clock can be set to follow its own hours, minutes and seconds, given custom names, set to use Daylight Savings, or set to be relative to your own location or GMT.

Time to Event

Time to Event saves a future date and counts down the days, hours, minutes and seconds to it. This is a free app available for download on the Apple App Store.
